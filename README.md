# ①課題名
ChatGPT API活用の占いWebサービス「押上の母」

## ②課題内容（どんな作品か）
ChatGPT APIを活用したAI占い師による人生相談サービス。

【主な機能】
- AI占い機能：GPT-4による深い洞察と具体的なアドバイスの提供
- 詳細な分析結果：
  1. 現状とあなたの状態
  2. あなたの性格・本質
  3. 今回の本当のテーマ
  4. これからの流れ・未来の可能性
  5. 具体的なアドバイス
  6. 最後に一言

## ⑤工夫した点・こだわった点

### アプリの制作意図
- OpenAI APIを実践的に活用
- ユーザーごとに最適化した回答を行う占いという形でAIの可能性を体験

### 工夫した点

#### デザイン面
- タロット占いの世界観を表現
- ユーザビリティ
  - レスポンシブデザイン対応
  - カスタムスクロールバー

#### 機能面
- プロンプトエンジニアリング
  - 詳細な役割設定
  - 6つの観点からの多角的分析を指示
  - 専門家への相談が必要な場合の安全配慮
  
- エラーハンドリング
  - 必須項目チェック（名前・相談内容）
  - APIキー未設定の検出
  - API通信エラーの適切な表示
  - try-catch-finally構文による確実な状態管理
  
- UI/UXの配慮
  - ローディング表示

#### コード面
- 非同期処理の実装
  - async/await構文の使用
  - fetch APIによるHTTP通信
  
- システムプロンプトの活用
  - GPT-4の回答品質を高める詳細な指示
